📱 แอปคำนวณกุ้ง คู่/คี่ + เลขท้าย

ใช้ได้ทั้งกดเลือก และพูด/พิมพ์บรรยายลักษณะกุ้ง จากนั้นระบบจะคำนวณผล พร้อมบันทึกประวัติในเครื่องอัตโนมัติ

1) เลือกข้อมูลหลัก

ออกคู่
เลขท้ายที่เป็นไปได้: 0, 2, 4, 6, 8
เหตุผล: ค่าเริ่มต้น

2) พูดหรือพิมพ์ลักษณะกุ้ง

ตัวอย่าง: “ตัวใหญ่ หัวโต ก้ามขวาเล็กสั้น” หรือ “ไม่มีกร้าม ตัวยาวกลางธรรมดา”
สถานะ: พร้อมใช้งาน
คำที่ระบบพอจับได้: เล็ก / กลาง / ใหญ่ / หัวโต / สมบูรณ์ / ไม่สมบูรณ์ / ไม่มี / ซ้าย / ขวา / สองข้าง / ยาว / สั้น / เล็ก / ใหญ่ / ดิ้น / หมดแรง

3) บันทึกข้อมูล

4) สรุปผลสะสม

5) ประวัติที่บันทึก

เวลา ล็อต/รอบ รายละเอียด ผล เลขท้าย นน.
ข้อมูลจะเก็บไว้ในเครื่อง/เบราว์เซอร์นี้อัตโนมัติ
